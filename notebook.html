Command Line:

  • Start the Thrift Server to enable connections from power bi /opt/spark/sbin/start-thriftserver.sh
  • Start spark shell with the sql driver path /opt/spark/bin/spark-shell --driver-class-path /usr/share/java/sqljdbc42.jar
  • Reset the tables :load $path initialize_tables.scala
  • Initialize the tables :load $path load_data.scala
  • Load Dates :load $path process_dim_date.scala
  • Load Devices :load $path process_dim_device.scala
  • Load Location :load $path process_dim_location.scala
  • Load fact entry measure :load $path process_fact_measure.scala
  • Load fact readings :load $path process_fact_measurement.scala
